# PSP Platform - Complete Functional Documentation

# 1. Project Overview

The PSP (Payment Service Provider) Platform is a centralized system used to manage merchant onboarding, deposits, withdrawals, settlements, transactions, approvals, and account management.

The platform consists of multiple portals:

1. Super Admin Portal
2. Admin Portal
3. Merchant Portal
4. Deposit Operator Portal
5. Withdrawal Operator Portal

Each portal has different permissions and responsibilities.

---

# 2. User Roles

## Super Admin

Highest authority in the system.

Responsibilities:

* Create Admin accounts
* View all Admins
* View all Merchants
* Monitor system activities
* Reset Admin passwords
* Activate/Deactivate Admins
* View reports and audit logs

The Super Admin does not process deposits or withdrawals.

---

## Admin

Responsible for managing merchants and operational approvals.

Responsibilities:

* Create Merchant accounts
* Manage Merchant profiles
* Approve or reject merchant requests
* Review deposits
* Review withdrawals
* View transaction history
* Monitor merchant activities

---

## Merchant

Uses the platform to perform transactions.

Responsibilities:

* Create deposit requests
* Create withdrawal requests
* View balances
* View transaction history
* View notifications
* Manage profile

---

## Deposit Operator

Responsible for handling incoming deposits.

Responsibilities:

* View deposit requests
* Verify deposit payments
* Manage QR payments
* Verify UPI payments
* Approve deposits

---

## Withdrawal Operator

Responsible for processing withdrawal requests.

Responsibilities:

* View withdrawal requests
* Verify withdrawal balance
* Process payouts
* Update withdrawal status

---

# 8. Forgot Password Workflow

User Clicks Forgot Password
↓
Enter Registered Email
↓
OTP Sent To Email
↓
Verify OTP
↓
Enter New Password
↓
Confirm Password
↓
Password Updated
↓
Login With New Password

OTP Expiry:
15 Minutes

---

# 9. Deposit Workflow

Merchant Creates Deposit
↓
System Generates Reference ID
↓
Merchant Makes Payment
↓
Admin Reviews
↓
Deposit Operator Verifies
↓
Approved
↓
Balance Updated

# 10. Withdrawal Workflow

Merchant Requests Withdrawal
↓
Admin Reviews
↓
Withdrawal Operator Processes
↓
Funds Released
↓
Completed

# 11. Reference Number Formats

Deposits:
DEP0000001

Withdrawals:
WDR0000001

Settlements:
SET0000001

Rules:

* Auto Generated
* Unique
* Sequential
* Reset To 1 When All Data Is Deleted
